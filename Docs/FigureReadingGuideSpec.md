# Figure Reading Guide Spec

The Figure Reading Guide explains how to interpret marker colors in a reference figure or simulated marker schematic.

The guide applies to two separate things:

- a real paper figure manually downloaded as `Docs/nihms313068f1.jpg` and copied/imported into `Assets/Art/ReferenceFigures/Matson2011_Figure1.png` as asset-based reference material;
- the simulator-generated marker pattern, which is an educational abstraction and not a real micrograph.

## Button

Button label:

```text
How to Read This Figure
```

## Guide Content

The guide should explain:

- SOX9 = green;
- FOXL2 = red;
- DAPI = blue;
- rows and columns should be compared for marker dominance, overlap, and condition-level differences;
- green-dominant pattern = more testis-like interpretation;
- red-dominant pattern = more ovary-like interpretation;
- mixed pattern = mixed or reprogramming-like interpretation.

The short guide embedded inside the reference modal should stay concise. The separate guide panel can include more explanatory text.

## Safety Wording

The guide must say that it is:

- an educational reading aid;
- mouse-model literature context;
- not medical or laboratory guidance;
- not direct human translation.

It should not provide wet-lab methods, staining protocols, animal procedures, or clinical interpretation.
