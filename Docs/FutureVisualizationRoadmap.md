# Future Visualization Roadmap

This roadmap describes the visual evolution of the simulator. Each phase must preserve the safety boundary: educational mouse developmental biology visualization only, with no wet-lab protocols, medical recommendations, or human-application guidance.

## Phase 1: Dashboard Foundation

Includes:

- current dashboard;
- parameter sliders;
- scenario presets;
- improved result explanation;
- score bars;
- layer toggle foundation;
- safer tooltip glossary foundation.

Goal:

```text
inputs
↓
pathway scores
↓
explanation
↓
basic visual layer switching
```

## Phase 2: Gonad and Endocrine Visualization

Includes:

- gonad anatomy visualization;
- undifferentiated, testis-like, ovary-like, mixed, and unstable states;
- simple endocrine arrows or flow;
- unitless hormone-like output bars;
- click-to-inspect organ details.

Goal:

```text
pathway scores
↓
gonad state
↓
abstract endocrine output
```

## Phase 3: Whole Mouse and Scenario Comparison

Includes:

- simplified whole mouse silhouette;
- highlighted gonad region;
- optional bloodstream overlay;
- side-by-side scenario comparison;
- synchronized timeline playback for compared scenarios.

Goal:

```text
current scenario vs comparison scenario
↓
organ state difference
↓
system-level educational context
```

## Phase 4: Optional 3D and Richer Animation

Includes:

- optional low-poly 3D mouse and organ assets;
- richer transitions between visual states;
- animated endocrine flow;
- Blender-generated assets if helpful;
- more polished camera and layer transitions.

This phase should happen only after the 2D educational simulator is useful and stable. 3D should add clarity, not realism for its own sake.

## Deferred Ideas

Possible later additions:

- maintenance/reprogramming view for DMRT1, FOXL2, SOX8, and SOX9;
- endocrine output curves;
- risk and uncertainty dashboard;
- concept-only tissue engineering visualization.

Deferred ideas must remain conceptual and non-procedural.
