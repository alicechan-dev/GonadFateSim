# Marker Hover Inspection Spec

The synthetic marker-pattern view supports hover-based local interpretation. This feature is an educational inspection overlay for the generated image; it is not an analysis tool for real microscopy.

## Interaction

When the pointer moves over the synthetic marker pattern:

- the hovered local area is sampled from the displayed visual state and pointer position;
- a soft curved outline appears around the inspected area;
- a tooltip follows the pointer;
- the tooltip hides when the pointer exits the marker image.

The outline should feel like a scientific inspection overlay, not a selection box.

## Hover Categories

### SOX9-like dominant zone

Used when the sampled local signal is green-dominant.

Tooltip wording:

```text
This region shows stronger simulated SOX9-like marker signal.
In this educational model, that suggests a more testis-like / Sertoli-like local identity.
```

### FOXL2-like dominant zone

Used when the sampled local signal is red-dominant.

Tooltip wording:

```text
This region shows stronger simulated FOXL2-like marker signal.
In this educational model, that suggests a more ovary-like / granulosa-like local identity.
```

### Mixed marker zone

Used when both simulated green and red marker signals are present.

Tooltip wording:

```text
This region contains both simulated SOX9-like and FOXL2-like signals.
In this educational model, that suggests a mixed or reprogramming-like local state.
```

### Nuclei-rich low-marker zone

Used when the sampled area is mostly DAPI-like nuclei signal with low simulated SOX9-like and FOXL2-like marker signal.

Tooltip wording:

```text
This region contains mostly DAPI-like nuclei signal with low visible SOX9-like and FOXL2-like marker signal.
This local area is not strongly emphasized by the current marker channels in the simplified model.
```

Do not describe blue-only areas as definite cell types or as performing a known biological role.

## Numeric Readout

Tooltips may show unitless local values:

- SOX9-like;
- FOXL2-like;
- DAPI-like.

These values are generated display signals, not measured fluorescence intensity.

## Scope Note

Every tooltip includes:

```text
Simulated interpretation, not real microscopy.
```

## Implementation Path

Current first-pass implementation:

- derive local UV from pointer position over the marker `RawImage`;
- estimate local SOX9-like, FOXL2-like, and DAPI-like signal from the displayed continuous visual state;
- classify the zone using conservative thresholds;
- draw a soft oval outline and UI tooltip.

Future improvement:

- add a hidden zone interpretation map or ID buffer generated by the shader;
- sample the ID buffer for cleaner zone boundaries;
- make the outline follow segmented tissue regions instead of a local oval.
